# Personal-CoachBot
Meet your personal Coach! An interactive command-line personal fitness coach built with Python. The program collects your personal data step by step and gives you tailored health metrics, workout plans, and a point-tracking system.

## Features
- Step-by-step onboarding — collects name, weight, height, age, and activity level
- BMI (Body Mass Index) calculator
- Daily calorie needs estimator based on user profile
- Personalized workout plan generator based on activity level
- Point tracking system — log your activity points and accumulate a total score

## Menu options
1. Check BMI
2. Check daily calorie needs
3. View a trining plan
4. Earn points
5. Exit

## How to run
1. Clone or download this repository
2. Open the `.ipynb` file in Jupyter Notebook
3. Run all cells
4. Follow the prompts in the output

## Technologies used
- Python 3
- Jupyter Notebook
- No external dependencies
